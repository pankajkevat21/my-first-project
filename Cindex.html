Chapter 2

Variables & Data Types

📖 Theory

Variables memory location hote hain jahan data store hota hai. Data type batata hai variable me kis type ka data rahega. 

#include <stdio.h>

int main(){

    int age = 20;

    float marks = 92.5;

    char grade = 'A';

    printf("%d\n",age);

    printf("%.2f\n",marks);

    printf("%c",grade);

    return 0;
}
💻 Output:

20
92.50
A
